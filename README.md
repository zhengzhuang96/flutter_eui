## 安装
在你的`pubspec.yaml`文件的dependencies节点下添加:
```
flutter_eui:
    git: https://github.com/zhengzhuang96/flutter_eui.git
```

导入:
```
import 'package:flutter_eui/flutter_eui.dart';
```
